## 更新内容

OpenClash Portal Android 首个公开版本发布。

本版本提供：
- 自动发现局域网内常见 OpenWrt 路由地址，减少手动输入成本
- 支持使用 `root + 密码` 登录 LuCI，并在应用内直接进入 OpenClash 页面
- 自动解析 OpenClash 状态接口，可快速跳转到 Zashboard 或 MetaCubeXD
- 支持手动覆盖 OpenClash、Zashboard、MetaCubeXD 的访问地址
- 支持会话持久化，重启应用后可恢复登录状态
- 支持 HTTPS 模式，并可按主机信任自签名证书

## 安装说明

- 下载并安装附件中的 APK
- 首次使用时，请确保手机已连接到目标 OpenWrt 所在局域网
- 若自动发现失败，可手动填写路由器地址或对应页面 URL

## 版本说明

- 当前版本号为 `1.0.0`
- 最低支持 Android 8.0（API 26）
